# Cargo Shipping Website

A web platform for cargo owners and truck owners/logistics companies to list, bid, and manage cargo shipments in real-time.

## Features
- User authentication (Cargo Owner, Truck Owner/Logistics Company)
- Cargo listing and auction system
- Real-time bidding and notifications
- User profiles, badges, and achievements
- Admin panel, FAQ, and more

## Setup
1. Clone the repository.
2. Set up Firebase project and update configuration in `assets/js/firebase.js`.
3. Open `index.html` in your browser to get started.

## Folder Structure
- `assets/css/` - Stylesheets
- `assets/js/` - JavaScript logic
- `assets/images/` - Images and icons
- `assets/sounds/` - Sound alerts
